# Backend - Supplier Performance System

## Database Schema
Located in `/database/schema.sql`

## Tables
- **suppliers** - Supplier information
- **purchase_orders** - Order tracking
- **order_items** - Order line items
- **deliveries** - Delivery records
- **quality_inspections** - Quality check results
- **supplier_kpis** - Calculated performance metrics
